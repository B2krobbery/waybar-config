# Waybar Config

Waybar configuration for Arch Linux with Hyprland.

## Screenshot

![Waybar Screenshot](waybar.png)

## Features

* Minimal top bar
* Workspace indicator
* Clock in the center
* System modules (volume, network, bluetooth, battery)

## Files

* `config` → Waybar module layout
* `style.css` → Waybar styling
